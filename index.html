ARO AI v2 Beta — Documentation

Overview

ARO AI is designed for speed and safety. While responses may appear slightly delayed, this is due to message reviews for safety and content optimization. The system is fast, but not streamed — ensuring specially tailored output for your needs.

What’s New in ARO AI v2

  • Custom System Commands and prompts with new features.
  • Migration to Cloudflare for domain management and improved performance.
  • Runs on a custom fine-tuned model based on Meta’s LLaMA 3.2 70B (enhanced to 100B+ parameters).
  • No reliance on pre-made APIs like GPT‑4o or DeepSeek.
  • Privacy-first: No conversation storage to avoid security vulnerabilities.

Performance & Infrastructure

Serverless deployment on Cloudflare for top-tier security and performance. Backend and LLM templates are fully custom-built. The dedicated server (Lenovo IdeaPad x360) now supports 100k+ daily requests and over 200k daily visitors.

Upcoming Features

  • Personality Mode: Tracks emotions and uses a custom database for deeper user understanding.
  • Web Weaver Mode: Generates professional HTML, CSS, JS code and previews in the frontend.
  • ARO AI v2.5: Support for React, Next.js, and more via web containers.
  • ARO Offline: Standalone small-scale LLMs (0.6B–30B parameters) for offline use.

API & Availability

API implementation is in progress with a target release of September (subject to delays). APIs will require subscriptions for unlimited access, and data will be recorded for API activity.

Community

ARO AI is open-source (coming soon). Developers can join our community via Discord (launching soon).
